Top 3 dos melhores sites que existem na web

  1. Google
  2. Youtube
  3. BeTrybe

Principais motivos para eu gostar deles

  • Google: Principal buscador trazendo muita facilidade nas minhas buscas de todo tipo; arquivos, imagens, videos, textos...
  • Youtube: Plataforma de vídeos gigante com todo tipo de conteúdo, quer aprender algo? você achará um video sobre o assunto.
  • BeTrybe: Curso com ótima qualidade, plataforma bem dinâmica, intercalando textos e vídeos para o conteúdo fixar da melhor forma.
